# SwiftFlow
Forge 1.16.5 inventory utility mod.

## What it does
- **Scroll** over a slot - swaps it with a matching hotbar slot
- **Shift + Scroll** over a slot - quick-moves the stack
- **Shift + Drag** across slots - quick-moves each slot you pass over
- **Home** - toggles the mod on/off (shows status in chat)

## Install
Drop `swiftflow-1.0.0-forge-1.16.5.jar` into `.minecraft/mods`.  
Requires Forge 1.16.5.
